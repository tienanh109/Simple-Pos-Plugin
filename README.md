# Simple-Pos-Plugin
A plugin that displays user coordinates using the /pos command for Minecraft PE 0.15.10 using PocketMine API 2.0.0!

## Info
- Show user coordinates only
- Command: /pos
- Language: Vietnamese
- API: 2.0.0
- Permission: All players
